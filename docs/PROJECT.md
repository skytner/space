# Space

## Описание

Приложение с интерактивной картой и профилями.

- Интерактивная карта (2D / 3D).
- Профили, привязанные к точкам на карте.
- Гостевой режим — просмотр без авторизации.

---

## Стек

| Часть   | Технология              |
|---------|-------------------------|
| Monorepo| Turborepo               |
| Frontend| Next.js 16 (`apps/web`) |
| Admin   | Next.js (`apps/admin`)  |
| API     | NestJS (`apps/api`)     |
| UI      | `@repo/ui` (React)      |

В планах: БД, Docker, nginx, CI/CD.

---

## MVP

1. **Лейаут** — оболочка приложения (шапка, область под карту, место под контент/профиль).
2. **Карта** — страница с картой (2D), панорама и зум. Переключатель 2D/3D (3D допустимо заглушить).
3. **Данные** — API отдаёт список точек; web запрашивает и отображает маркеры.
4. **Гостевой режим** — просмотр карты и маркеров без логина, без редактирования.
5. **Профиль** — клик по маркеру открывает страницу/модалку с данными точки (имя, описание); данные с API.
6. **Инфраструктура** — API + БД (SQLite или Postgres), при необходимости `docker-compose` (API + БД).

Вне MVP на первом этапе: авторизация, регистрация, админка, CI/CD.
